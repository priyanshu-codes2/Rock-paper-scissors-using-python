Project Description & Introduction Paragraph
    
    * The Terminal-Based Stone, Paper, Scissors Game is a Python mini-project designed to build strong programming logic.
    
    * It allows a user to play multiple rounds against the computer in a tournament format, 
      tracking scores continuously until a target score is reached.

    * This project demonstrates the practical use of fundamental Python concepts including:
      primitive variables, while loops for continuous execution, nested while loops for input validation,
      if-elif-else conditional structures, and basic input/output operations.
      
    * It serves as an excellent foundational project for beginners to practice logical flow and 
      error handling.
